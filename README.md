# starwars
Projeto para a disciplina Banco de Dados I - PCS3623 - Engenharia de Computação - POLI USP

1) Instalar MySQL
2) Instalar NodeJS
3) No MySQL crie um banco de dados com o nome "Starwars"
4) Clonar este repositório (git clone) para uma pasta na sua máquina
5) Abra um terminal na raíz do projeto e digite "npm install" para instalar os módulos nodejs do projeto
6) No arquivo ./starwars_bd/infraestrutura/conexao.js altere os campos "user" e "password" para acessar o seu MySQL
7) Para rodar o servidor digite "nodemon start" no terminal
8) Abra um navegador e digite "http://localhost:3000/starwars"
